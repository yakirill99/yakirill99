Отвечаю за полный жизненный цикл распределённых систем радиационного контроля на объектах транспортной безопасности.

### 🔑 Ключевые проекты и достижения:

#### 1. 🖥 Глобальная система управления (PowerShell TUI)
Разработал полнофункциональный консольный интерфейс (PSC.exe) для централизованного управления инфраструктурой:
- **Аналитика:** Сбор и парсинг статистик, анализ спектров, циклов калибровки и логов работы системы.
- **AI-интеграция:** Выявление зависимостей и аномалий через нейросеть, прогноз ремонта и рекомендации по эксплуатации.
- **Масштаб:** Структуризация и хранение логов на хосте (большие объемы телеметрии, тысячи устройств), поиск корреляций событий между узлами.
- **Автоматизация:** Конфигурация RPi через SSH, автоматизированная настройка IP-камер (Dahua, Hikvision) и АРМов (Win 11, Ubuntu).
- **Тестирование:** Стресс-тесты и анализ отказоустойчивости комплекса.

#### 2. 📡 Алгоритм локализации источников (TCP/IP Cluster)
Реализовал логику работы кластера из 20+ устройств на станциях метро:
- **Проблема:** При сильном источнике излучения срабатывали все датчики станции, блокируя входы.
- **Решение:** Алгоритм обмена данными по TCP/IP между устройствами. Определение пары с максимальной счётностью в реальном времени.
- **Результат:** Точная локализация источника до конкретного турникета. В логи пишется общая тревога, но система блокирует только проблемную зону.

#### 3. 🧠 AI для промышленного анализа
Внедряю нейронную сеть для конвейерного анализатора:
- **Задача:** Идентификация состава руды в потоке.
- **Роль:** Разработка и интеграция модели для автоматизации контроля качества сырья.
